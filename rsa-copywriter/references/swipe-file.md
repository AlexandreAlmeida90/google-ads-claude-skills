## WRITING PRINCIPLES

1. **Be specific, not vague** — "4.9 Stars From 5,000+ Reviews" beats "Highly Rated"
2. **Lead with the benefit, support with the feature** — "Whiten Teeth in 14 Days" (benefit) → "With Activated Charcoal Formula" (feature)
3. **Mirror the search intent** — the main keyword should feel like it's answering the query, not shoehorned in
4. **One idea per headline** — Google mixes headlines dynamically, so each must make sense alone
5. **Create urgency without being desperate** — "Sale Ends Sunday" > "Act Now!!!"
6. **Use numbers** — they build credibility and stand out ("60 Billion CFU" > "Probiotic-Rich")
